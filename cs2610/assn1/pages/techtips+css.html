HTML Elements CheatSheet

  1. <html>

    Defines the body of the document, contains all the visible content of the document.
    • Display Default: Block
    • Attributes Used: lang="en"
    • Accepted Contents: should countain both a <head> and <body>

  2. <head>

    Container for metadata and should be the first child of the <html> tag
    • Display Default: Neither
    • Attributes Used: None
    • Accepted Contents: <title>, <style>, <meta>, <link>, <script>, and <base>

  3. <link>

    Used to link external resources (like stylesheets) to you document, should be places in the <head>
    • Display Default: neither
    • Attributes Used: rel, href
    • Accepted Contents: Empty tag

  4. <title>

    Represents the title of the document (the name you'll see as the title to the tab). Should be places in the <head>
    • Display Default: neither
    • Attributes Used: None
    • Accepted Contents: Plain Text

  5. <style>

    Contains style information about the document in the from of CSS. Should be places in the <head>
    • Display Default: neither
    • Attributes Used: None
    • Accepted Contents: Content should be valid CSS syntax

  6. <body>

    Acts as the body of the document. All visible content should be placed within this tag. Should be the second child of the <html> tag
    • Display Default: Block
    • Attributes Used: None
    • Accepted Contents: Flow Content

  7. <h1>

    Denotse a top-level header for a section, there should be only one per page
    • Display Default: Block
    • Attributes Used: None
    • Accepted Contents: Phrasing Content

  8. <div>

    Generic container fod dividing content.
    • Display Default: Block
    • Attributes Used: class, id
    • Accepted Contents: Flow Content

  9. <p>

    Tag to represent a paragraph
    • Display Default: Block
    • Attributes Used: None
    • Accepted Contents: Phrasing Content

  10. <ul>

    Represents an unordered list of items
    • Display Default: Block
    • Attributes Used: class
    • Accepted Contents: <li> elements

  11. <ol>

    Represents an ordered list of items
    • Display Default: Block
    • Attributes Used: class
    • Accepted Contents: <li> elements

  12. <li>

    An Item in a list of items. Must be the child of either a <ul> or <ol>
    • Display Default: Block
    • Attributes Used: class
    • Accepted Contents: Flow Content

CSS Selectors Cheat Sheet

Basic Selesctors

  1. element

    The element selector is use to select an html tag 
        <div>
      ...
    </div>

    div {
      ...
    }

  2. #id

    The ID selections is used to uniquley identify a single element in an HTML document. 
        <div id="cool-id">
      ...
    </div>

    #cool-id {
      ...
    }

  3. .class

    The class selector is used to select all elements with the given attribute 
        <div class="cool-class">
      ...
    </div>

    <span class="cool-class">
      ...
    </span>

    .cool-class {
      ...
    }

Pseudo-selectors

  1. selector:nth-child(n)

    Selects the nth child of the given selector

  2. a:visited

    Selects <a> tags that have already been clicked on by the user

  3. selector:hover

    Selects an element when the user is hovering over the element with the mouse

Compound Selectors

  1. foo bar

    Selects a foo that is below bar somewhere in the DOM

  2. foo > bar

    Selects a bar that is a direct child of a bar

  3. .foo.bar

    Selects an element that has both the classes foo and code bar

CSS Properties Cheat Sheet

  1. background-color

    Changes the color of the background of the selected element
    Possible Values: white, black, #fff

  2. color

    Changes the color of the text
    Possible Values: white, black, #fff

  3. margin

    Used to create space outside of the border of the element
    Possible Values: 10px, 1em, 2.4rem

  4. padding

    Used to create space between an element's content and it's borer
    Possible Values: 10px, 1em, 2.4rem

  5. font-family

    Used to define the font-family that the element is to use
    Possible Values: Arial, monospace, sans-serif

  6. font-size

    Used to change the size of the element's text content
    Possible Values: 10px, 1em, 2.4rem

  7. border-radius

    Used to round the borders of an element
    Possible Values: 10px, 1em, 2.4rem

  8. text-decoration

    Used to give special decorations to some text
    Possible Values: underline, line-through, none

  9. display

    Used to change the way in which an element is displayed
    Possible Values: block, inline, flex

  10. text-decoration

    Used to change the way that the text is displayed
    Possible Values: italic, solid, dashed
